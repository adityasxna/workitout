## Getting Started

Visit [workitout.vercel.app](https://workitout.vercel.app) ‚Äî no installation or account needed.

### Run Locally

```bash
git clone https://github.com/alsiam/workitout.git
cd workitout
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS + shadcn/ui
- **State / Storage:** React Context + localStorage
- **Language:** TypeScript
